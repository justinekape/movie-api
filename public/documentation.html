MyFlix API

This is a REST API for an application called "myFlix" that interacts with a database that stores data about different movies.

Status Codes

200 "OK"
201 "Created"
400 "Bad Request"
500 "Internal Server Error"

List of All Movies

URL: /movies

Method: GET

Description: This endpoint allows users to view a list of all the movies.

Example
Request: localhost:8080/movies
Response: array containing all the movies available.

Movie Details by Title

URL: /movies/:title

Method: GET

Description: This endpoint returns data about a movie including description, genre, and director.

Example
Request: localhost:8080/movies/Spirited%20Away
Response:

A co-founder of Studio Ghibli, he has attained international acclaim as a masterful storyteller and creator of Japanese animated feature films, and is widely regarded as one of the most accomplished filmmakers in the history of animation.", "birth": "1941", "death": "-" }, "imageUrl": "https://www.imdb.com/title/tt0245429/mediaviewer/rm4207852801/?ref_=ext_shr_lnk", "featured": true }

Genre

URL: /movies/genre/:genreName

Method: GET

Description: This endpoint returns data about a genre.

Example
Request: localhost:8080/movies/genre/Animation
Response:
{ "name": "Animation", "description": "Animation is a genre in which the film's images are primarily created by computer or hand and the characters are voiced by actors." }

Director Information

URL: /movies/directors/:directorName

Method: GET

Description: This endpoint returns data about a director by name.

Example
Request: localhost:8080/movies/directors/Kevin%20Ko
Response:
{ "name": "Kevin Ko", "bio": "Taiwanese filmmaker Kevin KO came to prominence for a series of short films he made in college, including horror THE PRINT that screened at festivals in Japan and South Korea.", "birth": "1983", "death": "-" }

Add User

URL: /users

Method: POST

Description: This endpoint allows new users to register and assigns them a unique id.

Example
Request: localhost:8080/users
Response:
{ "name": "kim", "favoriteMovies": [], "id": "25814eb9-266c-4e3e-b814-4aaf5d717df0" }

Allow Users to Update Information

URL: /users/:id

Method: PUT

Description: This endpoint allows users to update their user info.

Example
Request: localhost:8080/users/1
Response:
{ "id": 1, "name": "kaul hilo", "password": "1234", "email": "kaulhilo@yahoo.com", "birthday": "1987-01-01", "favoriteMovies": [] }

Add Movie to Favorites

URL: /users/:id/:movieTitle

Method: POST

Description: This endpoint allows users to add a movie to their list of favorites. Once the movie has been added, a confirmation message will appear.

Example
Request: localhost:8080/users/3/Spirited%20Away
Response:
{ id: 3, name: 'anden', password: '0987', email: 'emoryanden@gmail.com', birthday: "1990-01-01", favoriteMovies: ["Spirited Away"] },

Message: Spirited Away has been added to anden's favorites

Remove Movie From Favorites

URL: /users/:id/:movieTitle

Method: DELETE

Description: This endpoint allows users to remove a movie from their list of favorites. Once the movie has been removed, a confirmation message will appear.

Example
Request: localhost:8080/users/3/Spirited%20Away
Response:
{ id: 3, name: 'anden', password: '0987', email: 'emoryanden@gmail.com', birthday: "1990-01-01", favoriteMovies: [] },

Message: Spirited Away has been removed from anden's favorites

Delete User

URL: /users/:id

Method: DELETE

Description: This endpoint allows exisitng users to deregister. Once their account has been deleted, a confirmation message will appear.

Example
Request: localhost:8080/users/2
Response: Message: shae has been deleted.
