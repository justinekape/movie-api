MyFlix API

This is a REST API for an application called "myFlix" that interacts with a database that stores data about different movies.

Status Codes

200 "OK"
201 "Created"
400 "Bad Request"
500 "Internal Server Error"

List of All Movies

URL: /movies

Method: GET

Description: This endpoint allows users to view a list of all the movies.

Example
Request: localhost:8080/movies
Response: array containing all the movies available.

Movie Details by Title

URL: /movies/:title

Method: GET

Description: This endpoint returns data about a movie including description, genre, and director.

Example
Request: localhost:8080/movies/The%20Lord%20of%20the%20Rings
Response:
{ "title": "The Lord of the Rings", "genre": { "name": "high fantasy", "description": "High fantasy, or epic fantasy, is a subgenre of fantasy defined by the epic nature of its setting or by the epic stature of its characters, themes or plot." }, "director": { "name": "Peter Jackson", "bio": "Sir Peter Robert Jackson ONZ KNZM is a New Zealand film director, screenwriter and producer.", "birth year": "October 31, 1961" } }

Genre

URL: /movies/genre/:genreName

Method: GET

Description: This endpoint returns data about a genre.

Example
Request: localhost:8080/movies/genre/high%20fantasy
Response:
{ "name": 'high fantasy', "description": 'High fantasy, or epic fantasy, is a subgenre of fantasy defined by the epic nature of its setting or by the epic stature of its characters, themes or plot.' },

Director Information

URL: /movies/directors/:directorName

Method: GET

Description: This endpoint returns data about a director by name.

Example
Request: localhost:8080/movies/directors/Peter%20Jackson
Response:
{ "name": 'Peter Jackson', "bio": 'Sir Peter Robert Jackson ONZ KNZM is a New Zealand film director, screenwriter and producer.', "birth year": 'October 31, 1961' }

Add User

URL: /users

Method: POST

Description: This endpoint allows new users to register and assigns them a unique id.

Example
Request: localhost:8080/users
Response:
{ "name": "Mary", "favoriteMovies": [], "id": "1c512b76-43e5-4de5-bea0-d4e98ea14428" }

Allow Users to Update Information

URL: /users/:id

Method: PUT

Description: This endpoint allows users to update their user info.

Example
Request: localhost:8080/users/1
Response:
{ "id": 1, "name": "Kim Updated Name", "favoriteMovies": [ "Venom" ] }

Add Movie to Favorites

URL: /users/:id/:movieTitle

Method: POST

Description: This endpoint allows users to add a moive to their list of favorites. Once the movie has been added, a confirmation message will appear.

Example
Request: localhost:8080/users/1/Coda
Response:
{ "id": 1, "name": "Kim Updated Name", "favoriteMovies": [ "Venom", "Coda" ] }

Message: Coda has been added to Kim Updated Name's favorites

Remove Movie From Favorites

URL: /users/:id/:movieTitle

Method: DELETE

Description: This endpoint allows users to remove a movie from their list of favorites. Once the movie has been removed, a confirmation message will appear.

Example
Request: localhost:8080/users/2/Parasite
Response:
{ "id": 2, "name": 'Harry', "favoriteMovies": [ "V for Vendetta" ] },

Message: Parasite has been removed from Harry's favorites

Delete User

URL: /users/:id

Method: DELETE

Description: This endpoint allows exisitng users to deregister. Once their account has been deleted, a confirmation message will appear.

Example
Request: localhost:8080/users/2
Response: Message: Harry has been deleted.
