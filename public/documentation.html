MyFlix API

This is a REST API for an application called "myFlix" that interacts with a database that stores data about different movies.

Status Codes

200 "OK"
201 "Created"
400 "Bad Request"
500 "Internal Server Error"

List of All Movies

URL: /movies

Method: GET

Description: This endpoint allows users to view a list of all the movies.

Example
Request: localhost:8080/movies
Response: array containing all the movies available.

Genre

URL: /genre/:Name

Method: GET

Description: This endpoint returns data about a genre.

Example
Request: localhost:8080/genre/Animation
Response:
{ "Name": "Animation", "Description": "Animation is a genre in which the film's images are primarily created by computer or hand and the characters are voiced by actors." }

Director Information

URL: /director/:Name

Method: GET

Description: This endpoint returns data about a director by name.

Example
Request: localhost:8080/director/Kevin%20Ko
Response:
{ "name": "Kevin Ko", "bio": "Taiwanese filmmaker Kevin KO came to prominence for a series of short films he made in college, including horror THE PRINT that screened at festivals in Japan and South Korea.", "birth": "1983", "death": "-" }

All Users

URL: /users

Method: GET

Description: This endpoint returns data about all users.

Example
Request: localhost:8080/users
Response: An array containing all users currently registered.

Add User

URL: /users

Method: POST

Description: This endpoint allows new users to register and assigns them a unique id.

Example
Request: localhost:8080/users
Response:

Add Movie to Favorites

URL: /users/:Username/movies/:MovieID

Method: POST

Description: This endpoint allows users to add a movie to their list of favorites.

Example
Request: localhost:8080/users/hilo/movies/630d22c5f9d2709917d1faf7
Response:
{ "_id": "630e5694940711d7c924621e", "Username": "hilo", "Password": "1234", "Email": "kaulhilo@yahoo.com", "Birthday": "1987-01-01T00:00:00.000z" "FavoriteMovies": [ "630e702373d830bca8fd52ac", "630e4bcb940711d7c9246218", "6304ab9940711d7c9246217", "630d22c5f9d2709917d1faf7" ] }

Remove Movie From Favorites

URL: /users/:Username/movies/:MovieID

Method: DELETE

Description: This endpoint allows users to remove a movie from their list of favorites.

Example
Request: localhost:8080/users/hilo/movies/630d22c5f9d2709917d1faf7
Response:
{ "_id": "630e5694940711d7c924621e", "Username": "hilo", "Password": "1234", "Email": "kaulhilo@yahoo.com", "Birthday": "1987-01-01T00:00:00.000z" "FavoriteMovies": [ "630e702373d830bca8fd52ac", "630e4bcb940711d7c9246218", "6304ab9940711d7c9246217", ] }

Delete User

URL: /users/:Username

Method: DELETE

Description: This endpoint allows exisitng users to deregister. Once their account has been deleted, a confirmation message will appear.

Example
Request: localhost:8080/users/newUser
Response: Message: newUser has been deleted.
